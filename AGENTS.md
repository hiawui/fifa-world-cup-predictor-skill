# Repository Instructions

## Skill Maintenance

- Treat `world-cup-match-predictor/` in this repository as the source of truth for the World Cup predictor skill.
- When updating the skill, edit or merge changes into the local `world-cup-match-predictor/` directory first.
- Before syncing, compare the local copy with `~/.codex/skills/world-cup-match-predictor/`. If the installed copy has useful newer changes, merge them back into the local copy first.
- After validating the local copy, sync it to `~/.codex/skills/world-cup-match-predictor/`.
- Avoid editing only the installed copy; that makes the repository version stale.
